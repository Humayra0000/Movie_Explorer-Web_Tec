# Movie Info Explorer

Project Overview
Movie Info Explorer is a web-based application developed for a Web Technologies course. This project allows users to explore detailed movie information using data fetched from a movie API. Users can search for movies by title and view metadata such as posters, ratings, release dates, genres, and more. The application demonstrates dynamic data fetching, API integration, and responsive UI designâ€”core concepts in modern web development.

Key Features
- Search Movies: Find movies by title and view real-time results  
- Movie Details: Display movie poster, summary, release date, genres, rating, and other key info  
- API Integration: Uses an external movie data API for real data  
- Responsive UI: Designed to work well on desktops and mobile devices  
- Interactive Frontend: JavaScript-powered dynamic interface for better user experience

Tech Stack
- Frontend: HTML, CSS, JavaScript  
- API: A public movie data API (e.g., OMDb API)  
- Tools: Git & GitHub for version control
